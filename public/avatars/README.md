# Avatar Videos

Place your agent avatar videos in this directory:

- `machina-avatar.mp4` - Video for the Machina goal-achievement agent
- `general-avatar.mp4` - Video for the general assistant agent
- `user-avatar.mp4` - Video for user avatar (optional)

## Video Requirements:
- Format: MP4
- Recommended size: 200x200px or 1:1 aspect ratio
- Duration: 2-10 seconds for looping
- File size: Under 5MB for optimal loading

If videos are not present, the system will automatically fall back to gradient avatars with emoji icons.
